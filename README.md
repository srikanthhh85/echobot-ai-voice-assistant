# 🎙️ EchoBot – AI Voice Transcription & Translation Assistant

EchoBot is a modern web application that converts speech into text in real time using the Web Speech API. It also supports multilingual translation, text export, and an intuitive user interface.

---

## 🚀 Features

- 🎤 Real-time Speech-to-Text
- 🌍 Multi-language Voice Recognition
- 🔄 Instant Text Translation
- 📋 Copy Transcribed Text
- 💾 Download as Text File
- 🗑️ Clear Transcription
- 📱 Responsive UI
- ⚡ Fast Browser-Based Processing

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Web Speech API
- Google Translate API
